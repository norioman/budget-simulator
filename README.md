# 予算シミュレーター

年間の売上・コスト計画から利益を算出し、貯金箱（設備・内装・試す・内部留保）への配分と実績を管理できるシンプルな予算管理ツールです。

🔗 **公開ページ**: https://norioman.github.io/budget-simulator/

## 使い方

1. **年間計画**: 今年の売上とコストの予測を入力
2. **配分**: 利益の何%を各貯金箱に入れるかスライダーで決定（残りは自動で内部留保へ）
3. **初期残高**: 現在の貯金を最初に一度だけ入力
4. **目標額**: 各貯金箱の目標額を設定
5. **予実管理**: 経過月数と実績の累計を更新して着地見込みを確認

### ルール

- 🎉 **クリア**: 目標を設定したすべての貯金箱が目標額に到達
- 💀 **ゲームオーバー**: 内部留保が0以下になる
- 達成判定は常にその時点の残高が目標額の100%以上かで決まる（使って減れば未達に戻る）
- 内部留保は常に目標を維持している必要がある
- 実績が黒字なら配分どおり積み上がり、赤字なら内部留保が全額吸収する

3〜5年でのクリアを目指す設計です。

## 技術構成

- Vanilla HTML / CSS / JavaScript（フレームワーク不使用、単一ファイル）
- データはブラウザの `localStorage` にのみ保存（サーバー通信なし）

## ローカルで開く

`index.html` をブラウザで直接開くだけで動作します。
